# VisualMasterPro 更新日志

本文档采用中文软件更新日志格式，记录 VisualMasterPro（雪原Ai增强引擎）的主要功能变化、优化内容、修复问题、架构升级与已知问题。

## V0.3 Quality Core

### 新增功能

- 新增 Text Clarity Engine，用于疑似文字区域局部清晰增强。
- 新增 Edge Safe Enhance，用于真实边缘安全增强。
- 新增 Structure Recovery，用于建筑、文物、远景等结构恢复。
- 新增 Highlight Protection，用于灯光、玻璃反光和展柜高光保护。
- 新增 Image Type Detector，用于识别 text_poster、architecture、artifact、portrait_indoor、ink_gray 和 general。
- 新增 Quality Compare，用于评估清晰度、文字、边缘、结构、纹理、噪声和色彩一致性。

### 优化内容

- fidelity 管线从“先放大再轻锐化”改为“先识别、先修复、局部增强、再放大、再轻补偿”。
- 输出前增加原图色彩锁定，降低自动变暖、自动 HDR 和色彩漂移风险。
- 批量日志会记录“有效清晰增强”或“伪高清 / 无效增强”质量标记。

### 修复问题

- 修复高光、反光区域容易被误增强的问题。
- 修复白底文字区域容易被误判为高光保护区域的问题。
- 降低全局锐化导致文字白边、黑边和边缘光晕的风险。

### 架构升级

- 新增 `engine/analysis/quality_compare.py`。
- 新增 `engine/analysis/image_type_detector.py`。
- 新增 `engine/analysis/edge_classifier.py`。
- 新增 `engine/algorithms/edge_safe_enhance.py`。
- 新增 `engine/algorithms/structure_recovery.py`。
- 新增 `engine/algorithms/highlight_protection.py`。

### 已知问题

- 当前仍为传统 CV 画质恢复方案，不是生成式超分辨率模型。
- 极小文字如果原图信息已经完全丢失，只能提升边界稳定性，无法凭空生成真实文字。
- 文物复杂纹理和远景结构仍需要后续引入更强的结构识别与质量评估数据集。

## V3.7

### 新增功能

- 规划商业签名引擎。
- 规划品牌视觉预设系统。
- 规划 Premium Presets。
- 规划批量商业化处理能力。

### 优化内容

- 强化各 mode 的专属视觉风格。
- 强化品牌色彩、光影、材质和细节节奏的统一控制。

### 修复问题

- 计划解决不同 mode 视觉差异不够明显的问题。
- 计划减少单纯依赖参数强度造成的风格相似问题。

### 架构升级

- 从 mode pipeline 升级为 mode、signature、preset 组合系统。
- 为创作者版、工作室版、企业版预留商业功能边界。

### 已知问题

- 需要更完整的行业参考图集。
- 需要更稳定的材质识别与场景判断。
