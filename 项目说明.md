# VisualMasterPro

## 商业级 AI 视觉增强系统

VisualMasterPro（雪原Ai增强引擎）是一个面向中国商业视觉行业的高端图像增强与品牌视觉处理系统。

它不是普通的图片清晰化脚本，而是面向 AI 视觉创作者、商业设计师、品牌视觉工作室和电商视觉团队的商业级视觉质量引擎。

## 项目定位

VisualMasterPro 的核心目标是将 AI 生成图、商业素材图和视觉草图，处理为更干净、更高级、更有商业交付感的视觉终稿。

核心方向：

- 商业视觉增强。
- 品牌质感统一。
- AI 高级图像优化。
- 原图忠实增强。
- 批量图片处理。

## 核心功能

### 商业签名引擎

为不同商业视觉类型建立独立视觉签名，例如化妆品、食品、奢侈品、人像、建筑、PPT 商业封面和品牌 KV。

### AI 图像增强

针对 AI 生成图常见问题进行处理，包括 AI 脏感、diffusion 纹理、假 HDR、高频污染和电子锐边。

### 高级锐化系统

不是简单全局锐化，而是通过高频控制、中频结构恢复和主体细节保护，避免出现脏边、毛刺和假细节。

### 商业级光影系统

控制高光、阴影、空气感和光影过渡，提升画面的商业摄影棚光感。

### 高端品牌色彩优化

统一画面色彩倾向，降低 AI 蓝、AI 橙、HDR 彩色污染和过饱和问题。

### 批量处理引擎

支持一次导入多张图片或整个文件夹，自动过滤非图片文件，单张失败不影响后续图片。

### 高分辨率输出系统

支持高分辨率商业输出方向，为 4K、8K 和专业交付做技术准备。

### 品牌视觉预设系统

用于沉淀常用增强参数，让不同类型图片保持稳定、可复用的处理效果。

## 原图忠实增强原则

默认处理不得主动修改构图、不得主动改色、不得重塑画风。

核心方向：

- 忠实放大。
- 清晰度提升。
- 噪点控制。
- 结构修复。
- 细节保护。
- 原图风格保持。

## V0.3 Quality Core 画质核心

当前阶段核心目标是解决“图片尺寸变大但真实清晰度没有明显提升”的伪高清问题。

新的 fidelity 处理链强调：

- 先修复压缩损伤，再进行放大。
- 识别文字区域，只对疑似文字做局部清晰增强。
- 识别真实边缘，避免把噪点、JPEG 块、高光和反光当成细节增强。
- 恢复建筑线条、文物纹理、远景结构等真实结构信息。
- 保护灯光、玻璃反光和展柜高光，避免高光炸开。
- 输出前进行色彩一致性校准，避免自动变暖、自动 HDR 或明显改色。
- 通过 Quality Compare 评估清晰度、文字、边缘、结构、噪声和色彩一致性。

## 快速启动

启动图形化界面：

```powershell
python main.py --gui
```

或：

```powershell
python -m gui.app
```

命令行处理：

```powershell
python main.py --input "D:\Codex\04_Visual-Master-Pro\输入图片" --output "D:\Codex\04_Visual-Master-Pro\输出成品" --mode fidelity
```

默认正式输出：

```text
输出文件夹/原文件名_vmp_v03_4k.png
```

开发调试模式：

```powershell
python main.py --input "输入图片" --output "输出成品" --mode ai_commercial_kv --debug
```

调试模式会额外输出：

- JSON 质量报告。
- Markdown 质量报告。
- 前后对比图。
