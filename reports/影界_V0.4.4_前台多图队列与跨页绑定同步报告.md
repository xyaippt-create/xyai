## 三、DashboardPage 当前能力

### 1. 多图队列

`DashboardPage.jsx` 已恢复 `fileQueue` 数组，不再使用单图状态。

支持：

- 多选图片
- 多图拖拽
- 自动过滤非图片文件
- 队列线性表格显示
- 每张图独立状态

队列状态：

```text
queued     待处理
uploading  上传中
processing 处理中
completed  已完成
failed     失败
```

队列表格字段：

```text
文件名
输入尺寸
输出尺寸
处理模式
输出格式
当前状态
输出文件名
操作
```

### 2. 串行调度

点击“开启核心修复管线”后：

```text
第 1 张上传
→ 监听该图 SSE
→ 收到 [DONE]
→ 关闭 EventSource
→ 更新该图结果
→ 自动处理下一张
```

严格禁止并发，避免多图任务互相覆盖。

### 3. 坏图容错

如果某张图返回 400 或解码失败：

- 当前行标记为 `failed`
- 错误信息写入该行
- 队列继续处理下一张
- 不会卡死整个任务队列

### 4. 输出位置配置

新增 `Output Location Monitor` 区域。

显示：

```text
输出位置：默认输出目录
输出位置：自定义目录
当前输出目录：C:\Users\xyppt\Desktop\雪原Ai增强引擎\输出成品
```

操作：

```text
验证输出路径 → POST /api/output/validate
应用输出路径 → 应用到后续 FormData output_dir
打开输出目录 → POST /api/output/open
恢复默认目录 → 清空自定义 output_dir
```

若后端返回：

```text
输出路径已存在，但不是目录。
```

前台会在输入框下方以红字显示，不抛异常，不黑屏。

## 四、跨页数据绑定

### 1. 当前队列项绑定

每张图完成后独立保存：

```text

```

### 2. 对比页绑定

点击某行“查看对比”：

```text
当前行 → makeTaskConfig(item) → ImageSliderComparePage
```

对比页优先读取：

```text
task_result.final_output_url
```

并强制转成：

```text
http://localhost:8787/api/file/outputs/{filename}
```

### 3. 禁止本地盘符图片路径

`ImageSliderComparePage.jsx` 已加安全过滤：

```text
C:\...
D:\...
```

不会直接进入 `<img src>`。

### 4. 报告页绑定

点击某行“查看报告”：

```text
当前行 → task_report / debug_quality → QualityReportPage
```

`QualityReportPage.jsx` 已合并读取：

```text
task_report
debug_quality
task_result.debug_quality
```

防止后端部分质量字段落在 `debug_quality` 时页面显示为空。
