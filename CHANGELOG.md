# VisualMasterPro CHANGELOG

## 项目定位

VisualMasterPro（雪原Ai增强引擎）不是普通图片增强工具，而是面向商业视觉交付的 AI 商业视觉增强引擎。

核心目标不是“让图片更清晰”，而是“让 AI 图更像高级商业视觉”。

---

## V1.0：基础增强阶段

### 阶段目标

解决 AI 图片基础可用问题，实现从普通 AI 图到 4K 可交付素材的基础增强。

### 已实现能力

- 中文路径安全读写
- 输入文件夹批量处理
- 4K 宽度输出
- PNG 无损输出
- 基础降噪
- 基础锐化
- 初步通透感增强
- EXE 打包运行
- 桌面自动生成输入图片 / 输出成品文件夹
- 输出命名统一为“_雪原Ai·PPT设计.png”

### 主要问题

- 增强逻辑偏传统图像处理
- 容易出现锐化污染
- AI 脏感识别能力不足
- 对商业高级感理解不足
- 无法区分真实结构与 AI 假纹理
- 不具备光影重建能力

---

## V2.0：AI 污染抑制与规则驱动阶段

### 阶段目标

从“简单增强”升级为“规则驱动的 AI 商业视觉引擎”。

### 已建立架构

- engine 核心算法层
- modes 视觉模式系统
- rules 全局视觉规则库
- ai_noise_rules AI 脏感数据库
- material_rules 材质规则库
- visual_style_rules 商业视觉风格库
- configs 默认配置
- docs 项目说明
- tests 后续测试目录

### 已注册视觉模式

- ai_commercial_kv：AI 商业 KV 模式
- cosmetics：化妆品模式
- food：食品模式
- cinematic：电影感模式
- architecture：建筑模式
- ppt_business：PPT 商业模式

### 已建立核心方向

- AI 脏感控制
- 高频污染控制
- 微对比增强
- 材质通透感
- 商业 CG 质感
- 电影空气感
- 不同材质不同增强逻辑

### 当前问题

- 与 V1.0 相比，视觉质变还不够明显
- 仍然偏图像优化，而不是商业视觉重建
- 高频压制后，中频结构保护不足
- 输出图有时出现轻微软化
- 光影仍然保留 AI 直出感
- 商业摄影棚光感不足
- 色彩统一能力不足

---

## V3.0：商业视觉重建阶段（当前方向）

### 阶段目标

不再追求“更清晰”，而是追求“更高级”。

目标是让 AI 图从“AI 生成图”升级为“高级商业视觉终稿”。

### V3 第一阶段工程记录（2026-05-24）

已完成 VisualMasterPro 从“增强流程”到“视觉质量闭环”的第一步工程化：

- 新建 `engine/analysis/visual_analyzer.py`，用于识别高频污染、AI 脏感、锐化风险、中频结构、高光风险、色彩污染、空气感不足和商业高级感基础评分。
- 重建 `engine/analysis/quality_report.py`，支持 JSON 和 Markdown 两种质量报告。
- 新建 `engine/algorithms/ai_noise_control.py`，用于控制 diffusion 纹理、AI 颗粒、假 HDR、电子锐边和高频脏纹理。
- 新建 `engine/algorithms/mid_frequency_structure.py`，用于恢复产品体积、食物结构、建筑筋骨、人像皮肤中频和商业材质层次。
- 新建 `engine/algorithms/commercial_light.py`，用于高光压缩、阴影透气、柔光感、棚拍光感和光影过渡平滑。
- 改造 `engine/pipeline/orchestrator.py`，形成“读取图片 → 视觉分析 → 推荐 / 选择 mode → AI 污染控制 → 中频结构恢复 → 商业光影 → 输出图片 → 质量报告”的闭环。
- 改造 `main.py`，支持 `--input`、`--output`、`--mode`、`--report`。
- 保持中文路径兼容，并保留后续 EXE 打包路径。

## V3.4：Mode Pipeline System

目标：

让不同 mode 真正拥有不同商业视觉风格，而不是只通过 strength 参数产生轻微差异。

### 已完成

- 新增 `pipelines/` 目录，作为商业视觉模式处理管线层。
- 新增 `food_pipeline.py`、`cosmetics_pipeline.py`、`luxury_product_pipeline.py`、`cinematic_pipeline.py`、`architecture_pipeline.py`、`portrait_pipeline.py`、`ppt_business_pipeline.py` 和 `ai_commercial_kv_pipeline.py`。
- 不同 mode 开始调用不同处理步骤，例如食物光泽、蒸汽空气感、皮肤柔光、奢侈品反射控制、建筑空间结构、PPT 文字边缘保护等。
- `engine/pipeline/orchestrator.py` 不再直接把所有 mode 固定走同一套增强顺序，而是通过 `run_mode_pipeline()` 调用独立模式管线。

### 当前意义

VisualMasterPro 开始从“通用增强流程”升级为“商业视觉模式系统”。

## 雪原商业视觉标准 V1

### 1. 高频规则

禁止：

- AI 颗粒
- diffusion 纹理
- 锐化白边
- 假 HDR
- 数码高光
- AI 毛刺
- 电子锐边

### 2. 中频规则

必须：

- 有结构
- 有体积
- 有材质
- 有商业筋骨

禁止：

- 奶油化过头
- 糊化
- 结构丢失
- 全局柔化

### 3. 光影规则

必须：

- 柔
- 通透
- 商业摄影化
- 高光不炸
- 阴影不死黑
- 光影过渡自然

禁止：

- AI 直射白光
- 爆高光
- 假反射
- 硬 HDR 光

### 4. 空气规则

必须：

- 有呼吸感
- 有景深
- 有空间层次
- 主体与背景分离

禁止：

- 全画面一样清楚
- 远景过实
- 背景抢主体

### 5. 色彩规则

必须：

- 色彩统一
- 商业品牌感
- 克制
- 高级

禁止：

- HDR 彩色污染
- AI 蓝橙冲突
- 过饱和
- 电子色

### 6. 人像商业规则

必须：

- 皮肤干净
- 柔但不假
- 毛孔不过度
- 高光自然
- 眼神清晰
- 发丝克制
- 背景空气化

禁止：

- 塑料皮肤
- 假毛孔
- AI 睫毛
- 发丝爆炸
- HDR 皮肤
- 假高光

### 7. 产品商业规则

必须：

- 产品干净
- 材质通透
- 高光丝滑
- 反射有控制
- 背景服务主体
- 低噪声
- 柔锐平衡

禁止：

- 粗糙纹理
- 过度锐化
- 金属脏边
- 玻璃发灰
- 高光炸白
