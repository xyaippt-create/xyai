# VisualMasterPro

## 商业级 AI 视觉增强系统

VisualMasterPro（雪原Ai增强引擎）是面向中国商业视觉行业的高端图像增强与品牌视觉处理系统。

当前项目首页使用中文说明体系。完整项目说明请阅读：

[项目说明.md](项目说明.md)

## 核心方向

- 商业视觉增强。
- 品牌质感统一。
- AI 高级图像优化。
- 原图忠实增强。
- 批量图片处理。

## 当前版本

```text
VisualMasterPro V0.3
```

当前对外测试阶段版本统一为 `VisualMasterPro V0.3`。

## 原图忠实增强原则

默认处理不得主动修改构图、不得主动改色、不得重塑画风。

核心方向：

- 忠实放大。
- 清晰度提升。
- 噪点控制。
- 结构修复。
- 细节保护。
- 原图风格保持。

## V0.3 Quality Core

当前画质核心阶段重点解决“伪高清”问题。处理链已从简单 resize 与全局轻锐化，升级为分区域画质增强：

- 文字区域清晰度增强。
- 真实边缘安全增强。
- 建筑、文物、远景结构恢复。
- 压缩损伤修复。
- 高光与反光保护。
- 原图色彩锁定。
- 画质对比与“伪高清 / 无效增强”标记。

## 快速启动

启动图形化界面：

```powershell
python main.py --gui
```

或：

```powershell
python -m gui.app
```

命令行批量处理：

```powershell
python main.py --input "D:\Codex\04_Visual-Master-Pro\输入图片" --output "D:\Codex\04_Visual-Master-Pro\输出成品" --mode fidelity
```

默认正式输出：

```text
输出文件夹/原文件名_vmp_v03_4k.png
```
