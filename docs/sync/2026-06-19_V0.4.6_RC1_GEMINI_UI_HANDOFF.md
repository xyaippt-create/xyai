
Dashboard 主体现在是三栏 flex：

```text

中央：flex-1 h-full flex flex-col overflow-hidden
右侧：w-[320px] h-full flex flex-col gap-4
```

布局意图：

- 左侧只放导入、模式、输出文件夹；
- 输出文件夹在左侧底部 `mt-auto` 锁定；
- 中央队列卡片 `flex-1` 纵向撑满；
- 队列表格只在内部 `overflow-auto`；
- 中央底部执行条固定 `h-16`；
- 右侧顶部输出参数固定；
- 右侧底部 `Pipeline 状态映射` 用 `mt-auto` 锁底；
- 主视窗不产生意外全局滚动条。

## 7. Gemini 需要重点检查的视觉问题

请重点检查 1080P 与高 DPI 缩放环境：

```text
1920x1080
Windows 125% 缩放
Windows 150% 缩放
浏览器缩放 100%

```

重点确认：

- 左侧 `RESTORATION MODES` 不再压到输出文件夹；
- 输出文件夹始终在左栏底部；
- 中央多图处理队列填满可用高度；
- 中央不会出现大面积黑色断层；
- 队列表格多行时只在内部滚动；
- 右侧 `Pipeline 状态映射` 不溢出暗色底盘；
- 右侧文本长串能截断或内部滚动；
- 底部执行条不遮挡队列；
- 全局页面不出现非预期横向或纵向滚动条。

## 8. 视觉风格要求

保持：

- 低明度；
- 硬朗格栅；
- 商业工具感；
- 边界清晰；
- 信息密度适中。

避免：

- 游戏 HUD；
- 霓虹发光；
- 大面积渐变；
- 装饰性光球；
- 营销落地页风格；
- 过大标题；
- 卡片套卡片造成层级混乱。
