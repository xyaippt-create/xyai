# engine 核心算法模块

`engine` 是 VisualMasterPro 的长期核心。

## 子目录

| 目录 | 保存内容 |
|---|---|
| `algorithms/` | 单一职责算法模块。每个文件只解决一类视觉问题。 |
| `analysis/` | 图片分析和质检模块。未来用于自动判断模式和输出风险。 |
| `io/` | 图片读写、中文路径兼容、文件收集。 |
| `pipeline/` | 处理流水线，把多个算法按顺序组合成完整引擎。 |

## 代码原则

- 算法函数尽量纯粹：输入图片和参数，输出图片。
- 不在算法模块里写文件路径。
- 不在算法模块里写行业审美判断。
- 行业判断交给 `modes/` 和规则库。
- 处理顺序交给 `engine/pipeline/`。
