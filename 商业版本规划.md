# VisualMasterPro 商业版本规划

本文档用于规划 VisualMasterPro 的商业版本体系，为后续收费版本、商业授权、团队协作和企业级部署提供产品边界。

## 版本总览

| 能力 | 免费版 | 创作者版 | 工作室版 | 企业版 |
|---|---:|---:|---:|---:|
| 商业授权 | 不含 | 个人商业授权 | 工作室商业授权 | 企业商业授权 |
| 输出规格 | 2K | 4K | 8K | 定制规格 |
| GPU 支持 | 不支持 | 可选支持 | 支持 | 支持 |
| 批量输出 | 试用限制 | 标准批量 | 高级批量 | 企业队列 |
| 品牌预设 | 基础预设 | 高级预设 | 完整预设库 | 定制预设 |
| 云同步 | 不支持 | 可选 | 支持 | 私有化 / 托管 |
| API 能力 | 不支持 | 不支持 | 有限开放 | 完整开放 |
| 团队协作 | 不支持 | 不支持 | 支持 | 支持 |
| 优先支持 | 社群支持 | 标准支持 | 优先支持 | SLA 支持 |

## 免费版

### 面向人群

- 个人体验用户。
- 学习用户。
- 非商业 AI 图像创作者。

### 功能权限

- 基础图像增强。
- 限量 mode。
- 输出带限制。
- 不支持完整商业预设。

### GPU 支持

不支持。

### 商业授权

不包含商业授权。

### 批量输出

仅提供有限试用次数。

### API 能力

不开放。

### 云同步

不支持。

### 价格定位建议

免费，用于拉新、体验和基础传播。

## 创作者版

### 面向人群

- 商业设计师。
- PPT 设计师。
- AI 视觉创作者。
- 自媒体与小型内容团队。

### 功能权限

- 支持核心商业 mode。
- 支持 4K 输出。
- 支持标准高级预设。
- 支持小规模批量处理。

### GPU 支持

可选支持本地 GPU 加速。

### 商业授权

包含个人商业授权。

### 批量输出

支持常规项目批量输出。

### API 能力

不开放。

### 云同步

可选预设同步。

### 价格定位建议

适合月订阅或个人永久授权。

## 工作室版

### 面向人群

- 品牌视觉工作室。
- 电商视觉团队。
- 商业摄影后期团队。
- 企业市场部设计小组。

### 功能权限

- 支持完整 mode。
- 支持完整 Premium Presets。
- 支持项目级预设管理。
- 支持质量报告和对比输出。

### GPU 支持

支持。

### 商业授权

包含工作室级客户项目商业授权。

### 批量输出

支持高级批量处理、预设批量套用和项目输出队列。

### API 能力

开放有限本地自动化接口。

### 云同步

支持团队预设同步与项目设置同步。

### 价格定位建议

适合专业订阅、年度授权或团队席位授权。

## 企业版

### 面向人群

- 大型品牌方。
- 企业设计部门。
- 平台型内容生产团队。
- 需要内部视觉标准化的组织。

### 功能权限

- 支持私有化部署。
- 支持品牌规则定制。
- 支持企业级视觉规范。
- 支持权限、审计和流程集成。

### GPU 支持

支持工作站、服务器或私有算力部署。

### 商业授权

包含企业级商业授权。

### 批量输出

支持企业级批量队列、自动化任务和工作流集成。

### API 能力

完整开放 API。

### 云同步

支持私有云、托管云或本地化部署。

### 价格定位建议

适合年度合同、席位授权、项目授权和 API 调用量组合定价。
